# Contributing to Autonomous Orchestrator

We welcome contributions to the Autonomous Orchestrator project. To get started, please follow these steps:
1. Fork the repository: `git fork https://github.com/autonomous-orchestrator/autonomous-orchestrator.git`
2. Create a new branch: `git checkout -b <branch_name>`
3. Make changes and commit: `git add <file> && git commit -m '<commit_message>'`
4. Push changes: `git push origin <branch_name>`
5. Create a pull request: `git pull-request <branch_name>`
